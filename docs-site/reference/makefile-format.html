SYSTEM_PROMPT block

define SYSTEM_PROMPT
You are a helpful assistant.
endef

Tool blocks

# <tool>
# Description of the tool.
# @param NAME string Description
# </tool>
my-target:
	@echo "$$NAME"

Parameter access

Parameters are injected as environment variables. In recipes, use $$PARAM as the canonical form (for example $$NAME).

$(PARAM) also works for simple single-line values because Make imports environment variables.

Optional variables

You can define agent metadata and built-in-tool policy directly in the Makefile:

define DESCRIPTION
Specialist description shown by list_agent.
endef

DISABLED_BUILTINS = run_agent,validate_agent
# or: DISABLED_BUILTINS = all
