Minimal Agent Example

On this page

  • Makefile
  • Run

Makefile

define SYSTEM_PROMPT
You are a filesystem assistant.
endef

.PHONY: list-files

# <tool>
# List files in a directory.
# @param DIR string Directory path
# </tool>
list-files:
	@ls -la "$$DIR"

Run

uv run make_agent -f Makefile
