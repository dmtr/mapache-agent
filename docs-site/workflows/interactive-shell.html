Interactive Shell

On this page

  • When to use
  • Command
  • Tips

When to use

Use interactive mode for iterative tasks where you want back-and-forth conversation.

Command

uv run make_agent -f Makefile

Tips

Use /exit or /quit to leave the shell (or press Ctrl-D).

Useful shell commands: /help, /export, and /stats (token stats require memory enabled).
