Where settings live

Settings are stored in ~/.make-agent/<project-slug>/settings.yaml.

Sample settings.yaml

model: anthropic/claude-haiku-4-5-20251001
makefile: ./my-agent.mk
memory: true
reasoning_effort: low

Use --agent-model on the CLI when you want specialists called via run_agent to use a different model than the main agent.

Flag precedence

CLI flags override values from settings.yaml.
